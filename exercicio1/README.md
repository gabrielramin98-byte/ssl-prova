# Exercício 1 – Instalação do OpenSSL

## 📌 Comandos utilizados

```bash
sudo apt update
sudo apt install openssl -y
openssl version

O OpenSSL é uma ferramenta utilizada para trabalhar com criptografia e certificados digitais. 
Ele permite criar chaves, certificados e implementar segurança em conexões HTTPS.

Neste exercício foi realizada a instalação da ferramenta e verificação da versão.
